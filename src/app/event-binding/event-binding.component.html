Event Binding

Syntax for event binding is : ('event')="method()" e.g: (click)="onClick()"

DOM has abehavior called event bubbling or propogation, it means when there is an event on both perant

and child element the event will bubble up first rasing the event on child element then on Parent

to stop these we have method called ".stopPropogation()" on the event object.

Further there are several other properties like x and y cordinates for mouse on the event object.
