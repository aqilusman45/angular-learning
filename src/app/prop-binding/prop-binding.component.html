Property binding in angular

We need to learn about the difference between

HTML element and a DOM property

Now lets consider an input element

Attributes and properties are not same

Attributes are initialized in HTML

While properties are DOM properties

Attributes init DOM properties and they cannot be changed

Property values can change

We actually bind our data/variables to DOM property not the html attributes

it may seem like we are changing the value of value attribute

but that is not the case

If we want to bind a variable to an element to render dynamic data

We use the following syntax

['attribute property']="value"
